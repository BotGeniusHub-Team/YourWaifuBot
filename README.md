# YourWaifuBot
Telegram Waifu Bot

This bot will retrieve random waifu for you. Available commands are `/waifu` and `/shipgirl`.

Special thanks to [Reddy](http://twitter.com/Derpmia) and [Muaz](http://twitter.com/muaz_san) for helping with the list.
